# AudioDeviceSync
Do you often switch between headphones and speakers while playing WoW?
Are you tired of doing the whole Esc > Options > Audio > Output Device shuffle?
Well, we have a fix for you!
Simply set your Output Device to "System Default" in WoWs sound settings, and install this WA which will restart the sound system, thus picking up the new audio device, whenever you change the default audio device in your OS.

You could accomplish the same thing by running the macro `/run Sound_GameSystem_RestartSoundSystem()` whenever you change your default device in your OS, but why when you can automate that?
